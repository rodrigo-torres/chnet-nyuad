 System Requirements
 -------------------
 
 - Java Runtime Envirnment
 - A CAEN Digitizer with DPP PHA firmware
 - Linux kernel >= 2.4.x


 Installation notes
 ------------------
 
 - Install with:
     ./configure
     make
     make install #(as root)

 - Launch with:
     DPP-PHA_ControlSoftware
